# Donor Intelligence & Fundraising Optimization Platform

Senior Data Scientist Portfolio Project

## Overview
End-to-end fundraising analytics platform for organizations such as nonprofit, advocacy,
and campaign fundraising teams.

## Features
- Donor segmentation (RFM + clustering)
- Donor churn prediction
- Lifetime value prediction
- Donation forecasting
- NLP donor feedback intelligence
- Recommendation engine
- FastAPI prediction API
- MLflow experiment tracking
- Docker deployment structure

## Tech Stack
Python, SQL, Pandas, Scikit-learn, XGBoost, LightGBM, FastAPI, Docker, MLflow, NLP
